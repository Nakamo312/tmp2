# Байесовская классификация: обнаружение сигнала на фоне шума

## 1. Постановка задачи

Два класса наблюдений:

- **Класс Ω₀** («шум») — объект отсутствует, вектор средних **M₀ = (0, 0)ᵀ**
- **Класс Ω₁** («сигнал + шум») — объект присутствует, вектор средних **M₁ = (a, a)ᵀ**

Ковариационная матрица **одинакова** для обоих классов:

$$B = \begin{pmatrix} \sigma^2 & \eta \\ \eta & \sigma^2 \end{pmatrix}$$

Условия задачи:

- Равные априорные вероятности: $P(\Omega_0) = P(\Omega_1) = 0.5$
- Простейшая матрица штрафов:

$$C = \begin{pmatrix} c_{00} & c_{01} \\ c_{10} & c_{11} \end{pmatrix} = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$$

---

## 2. Байесовский классификатор

При равных априорных вероятностях и простейшей матрице штрафов решающее правило сводится к сравнению правдоподобий:

$$d(\mathbf{x}) = \arg\max_j f(\mathbf{x} \mid \Omega_j)$$

Для двух нормальных распределений с одинаковой ковариацией разность логарифмов правдоподобий:

$$\ln f(\mathbf{x}|\Omega_1) - \ln f(\mathbf{x}|\Omega_0) = (M_1 - M_0)^\top B^{-1} \mathbf{x} - \frac{1}{2}(M_1^\top B^{-1} M_1 - M_0^\top B^{-1} M_0)$$

---

## 3. Обратная матрица B⁻¹

Определитель:

$$\det B = \sigma^4 - \eta^2 = (\sigma^2 - \eta)(\sigma^2 + \eta)$$

Обратная матрица:

$$B^{-1} = \frac{1}{\sigma^4 - \eta^2} \begin{pmatrix} \sigma^2 & -\eta \\ -\eta & \sigma^2 \end{pmatrix}$$

Условие положительной определённости: $|\eta| < \sigma^2$.

---

## 4. Линейная решающая функция

Подставим $M_0 = (0,0)^\top$, $M_1 = (a,a)^\top$:

**Первое слагаемое:**

$$(M_1 - M_0)^\top B^{-1} = (a, a) \cdot \frac{1}{\sigma^4 - \eta^2} \begin{pmatrix} \sigma^2 & -\eta \\ -\eta & \sigma^2 \end{pmatrix} = \frac{a(\sigma^2 - \eta)}{\sigma^4 - \eta^2}(1, 1) = \frac{a}{\sigma^2 + \eta}(1, 1)$$

**Второе слагаемое:**

$$M_1^\top B^{-1} M_1 = \frac{2a^2}{\sigma^2 + \eta}, \qquad M_0^\top B^{-1} M_0 = 0$$

**Итоговая решающая функция:**

$$g(\mathbf{x}) = \frac{a}{\sigma^2 + \eta}(x_1 + x_2) - \frac{1}{2} \cdot \frac{2a^2}{\sigma^2 + \eta} = \frac{a}{\sigma^2 + \eta}(x_1 + x_2 - a)$$

---

## 5. Решающее правило

Класс **Ω₁** выбирается, если $g(\mathbf{x}) > 0$, то есть:

$$\boxed{x_1 + x_2 > a \implies \Omega_1, \qquad x_1 + x_2 \leq a \implies \Omega_0}$$

Порог классификации: $\theta = a$.

---

## 6. Распределение решающей статистики

Статистика $T = x_1 + x_2$ — линейная комбинация нормальных величин:

| Класс | Среднее T | Дисперсия T |
|-------|-----------|-------------|
| Ω₀ | 0 | $2(\sigma^2 + \eta)$ |
| Ω₁ | 2a | $2(\sigma^2 + \eta)$ |

Дисперсия суммы:
$$\mathrm{Var}(x_1 + x_2) = \sigma^2 + \sigma^2 + 2\eta = 2(\sigma^2 + \eta)$$

---

## 7. Вероятности ошибок

**Ошибка 1-го рода** (ложная тревога) — истинный класс Ω₀, решение Ω₁:

$$P(\text{ош} \mid \Omega_0) = P\!\left(T > a \mid \Omega_0\right) = 1 - \Phi\!\left(\frac{a}{\sqrt{2(\sigma^2+\eta)}}\right)$$

**Ошибка 2-го рода** (пропуск сигнала) — истинный класс Ω₁, решение Ω₀:

$$P(\text{ош} \mid \Omega_1) = P\!\left(T \leq a \mid \Omega_1\right) = \Phi\!\left(\frac{a - 2a}{\sqrt{2(\sigma^2+\eta)}}\right) = 1 - \Phi\!\left(\frac{a}{\sqrt{2(\sigma^2+\eta)}}\right)$$

Обе ошибки **равны** — следствие симметрии задачи при равных априорных вероятностях.

---

## 8. Суммарный риск (байесовский риск)

$$R = \frac{1}{2} P(\text{ош} \mid \Omega_0) + \frac{1}{2} P(\text{ош} \mid \Omega_1)$$

Поскольку оба слагаемых равны:

$$\boxed{R = 1 - \Phi\!\left(\frac{a}{\sqrt{2(\sigma^2 + \eta)}}\right)}$$

где $\Phi$ — функция стандартного нормального распределения.

Эквивалентная запись через Q-функцию ($Q(z) = 1 - \Phi(z)$):

$$R = Q\!\left(\frac{a}{\sqrt{2(\sigma^2 + \eta)}}\right)$$

---

## 9. Анализ поведения риска

Риск зависит от единственного безразмерного параметра:

$$t = \frac{a}{\sqrt{2(\sigma^2 + \eta)}}$$

### Зависимость от a (амплитуды сигнала)

| a | t | R |
|---|---|---|
| 0 | 0 | 0.5 (случайное угадывание) |
| малое | малое | близко к 0.5 |
| большое | большое | близко к 0 |

При $a \to \infty$: $R \to 0$ — идеальная классификация.

### Зависимость от σ² (дисперсии шума)

- При $\sigma^2 \uparrow$: $t \downarrow$, риск $R \uparrow$ — шум «засыпает» сигнал.
- При $\sigma^2 \to \infty$: $R \to 0.5$.

### Зависимость от η (ковариации компонент)

η входит в риск только через сумму $\sigma^2 + \eta$:

| η | Эффект |
|---|--------|
| η > 0 (положительная корреляция) | $\sigma^2 + \eta > \sigma^2$ → риск выше |
| η = 0 (независимые компоненты) | $R = 1 - \Phi\!\left(\frac{a}{\sqrt{2}\,\sigma}\right)$ |
| η < 0 (отрицательная корреляция) | $\sigma^2 + \eta < \sigma^2$ → риск ниже |
| η → −σ² | вырожденный случай, матрица B необратима |

**Физический смысл:** при положительной корреляции компоненты x₁ и x₂ несут меньше независимой информации о классе — эффективная размерность снижается, классификация ухудшается.

---

## 10. Связь с общей формулой (произвольная размерность n)

Для вектора размерности n с независимыми компонентами (η = 0) риск:

$$R = 1 - \Phi\!\left(\frac{a\sqrt{n}}{2\sigma}\right)$$

При n = 2, η = 0 оба выражения совпадают:

$$1 - \Phi\!\left(\frac{a\sqrt{2}}{2\sigma}\right) = 1 - \Phi\!\left(\frac{a}{\sqrt{2}\,\sigma}\right) = 1 - \Phi\!\left(\frac{a}{\sqrt{2(\sigma^2 + 0)}}\right) \checkmark$$

---

## 11. Итог

**Байесовский классификатор** при данных условиях — это пороговое устройство, вычисляющее сумму компонент наблюдаемого вектора и сравнивающее её с порогом $\theta = a$.

**Риск классификации:**

$$\boxed{R = 1 - \Phi\!\left(\frac{a}{\sqrt{2(\sigma^2 + \eta)}}\right)}$$

- Убывает с ростом $a$
- Убывает с уменьшением $\sigma^2 + \eta$
- Ковариация $\eta$ влияет через эффективную дисперсию суммы $2(\sigma^2 + \eta)$
